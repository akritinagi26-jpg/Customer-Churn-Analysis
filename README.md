# Customer Churn Analysis

## Overview
This project analyzes customer data to identify patterns leading to customer churn and suggests strategies to improve retention.

## Tools Used
- Excel
- Power BI
- Tableau
  
## Dataset
- 10,000+ customer records
- Includes demographics, usage, and subscription details

## Work Done
- Cleaned and processed raw data
- Performed exploratory data analysis
- Built Power BI dashboard

## Key Insights
- Low-engagement users are more likely to churn
- Monthly plans have higher churn rates
- Customer support impacts retention
